# Contributing

Desktop Icon Position is a source-available project, not an open-source project.
Before contributing, read [LICENSE](LICENSE) carefully.

## Before You Spend Time

- There is no guarantee that any issue, suggestion, or pull request will be reviewed, accepted, merged, or supported.
- For larger changes, contact Gustavo Lupatelli at gmlupatelli@gmail.com before doing substantial work.
- Do not submit confidential information, proprietary code, or anything you do not have the right to share.

If you do not agree to those terms, do not submit a contribution.

## Scope of Contributions

Helpful contributions include:

- bug reports with clear reproduction steps;
- small focused fixes;
- test improvements;
- documentation corrections.

Please keep changes narrow and avoid unrelated refactors.

## Coding Notes

- Follow the existing code style and project structure.
- Prefer minimal, targeted changes.
- Update tests or docs when they are directly affected by your change.
- Do not add third-party code unless you clearly identify its license and have permission to contribute it.
